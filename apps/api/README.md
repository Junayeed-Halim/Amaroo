# API App (NestJS scaffold)

Phase 1 backend scaffolding for Amaroo technical specification:

1. Auth module
2. Products module
3. Checkout/COD order flow
4. Courier integration (Steadfast-first)

The initial implementation is intentionally minimal and uses NestJS-style modules/controllers/services/entities so future iterations can swap in TypeORM/Redis/payment gateways without changing endpoint shapes.
