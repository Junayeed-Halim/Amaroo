# ROADMAP

## Phase 0 - Foundation (current scaffold)

- Monorepo structure for apps/services/packages/infrastructure
- Buyer web Next.js bootstrap with bilingual toggle baseline
- Service skeletons with contracts and placeholder routes
- Local dependency stack (PostgreSQL, Redis, Meilisearch)

## Phase 1 - Trust-critical MVP (Bangladesh)

- Account creation + phone verification
- Product catalog with Bengali/English metadata
- Cart/checkout with COD + SSLCommerz + bKash initial integration
- Courier booking MVP with Pathao + Steadfast
- Order tracking and push notifications

## Phase 2 - Merchant and operations scale

- Full seller dashboard workflows and analytics
- Admin KYC, dispute management, and risk controls
- Additional payment rails (Nagad, Rocket, AmarPay, cards)
- Additional couriers (REDX, Paperfly, Sundarban)

## Phase 3 - Compliance and resilience

- Enhanced fraud rules and suspicious activity review queues
- Data retention and audit policies aligned with Bangladesh operations
- Payment/dispute evidence traceability and reporting
- CDN/storage hardening with Cloudflare + S3/Wasabi governance
